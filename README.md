# AvatarPlayTrial
AvatarPlay是在Unity扩展的一个独立的动画系统。它拥有独立的动画格式，因此可以更容易和更灵活地扩展和定制，且更方便于跨平台。AvatarPlay支持动画重定向，不限于人形或非人形动画。你可以自己创建和定义Avatar，以及在unity内创建和编辑修改动画。

主要特性包括：
* 更小的动画体积
* 创建和编辑Avatar
* 编辑Avatar Mapped
* 创建和编辑动画
* 动画重定向，可定义Avatar，支持所有物体类型
* 动画平滑过渡
* Loop播放和RootMotion播放支持
* 帧命名和帧事件
* 完全的骨骼遮罩支持
* 在内存中压缩储存/加载动画
* 多层动画支持
* 动画预览工具
* 动画转换工具
* 动画录制烘培工具
* 用clipset管理和配置动画
* 完全开放源代码（使用c#）

- 因为优化的需要，暂不支持缩放动画
